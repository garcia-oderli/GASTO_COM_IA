🔒 COMO RESOLVER

O Chrome no celular exige HTTPS para liberar o microfone. Arquivos abertos via file:// são bloqueados.

① Netlify Drop — 30 segundos, grátis
Acesse app.netlify.com/drop, arraste o arquivo meetrec_v1.html e abra o link gerado no celular. Sem cadastro, funciona imediatamente.
② GitHub Pages — permanente
Crie repositório público → suba como index.html → ative GitHub Pages. URL HTTPS fixa.
③ Modo fallback — sem microfone
Grave com o gravador nativo do celular, depois carregue o arquivo de áudio no MeetRec para gerar a ata com IA.
⚠ Microfone bloqueado — página precisa de HTTPS.
Arraste o arquivo em app.netlify.com/drop → abra o link no celular.  ·  Outras opções
